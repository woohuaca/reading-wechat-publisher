# v0.1.0

`reading-wechat-publisher` is the first published version of the plugin.

## What This Release Covers

- turn a book title, note set, or reading draft into a structured markdown note
- reshape that note into a WeChat Official Account article
- generate Guizang-style cover and card guidance for stronger social presentation
- run local preview and preflight checks before WeChat publishing
- create WeChat drafts through the bundled Python scripts
- support Xiaohongshu / Rednote draft packaging with browser-assisted follow-through

## Included Skills

- `reading-strategy`
- `wechat-article-designer`
- `wechat-publisher`
- `reading-social-publisher`

## Practical Boundaries

- WeChat formal publish is intentionally not the default path
- Xiaohongshu still needs a manual image-upload step in the browser before automation continues
